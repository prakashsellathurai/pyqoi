# pyqoi
PyQoi is a python library for Encoding/Decoding the images in  “[Quite OK Image](https://github.com/phoboslab/qoi)” (qoi) format for fast, lossless image compression.

## Setup

 ```bash
 pip install -r requirements.txt
 ```

## Test

 ```bash
 python test.py
 ```
